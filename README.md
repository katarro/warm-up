# warm-up
Created with CodeSandbox

Programa desarrollado con la libreria de javascript React.js.

1. Para abrir clonar el repositorio previamente se debe tener instalado git hub en su ordenador.
2. En la terminal de comandos (cmd) ingresar: git clone "url del repositorio"
3. Ingresar el comando cd warm-up para acceder a la ruta.
4. Ingresar el codigo "install npm"
5. Ingresar el codigo "npm start" de esa manera se podra ver el programa en funcionamiento.
